REMOTE ACCESS:



SSH- remote computer protocol

OpenSSH - Linux remote client program
puTTy -	Windows remote access program

SSH Keys- key pairs that allow for secure SSH login between 2 remote systems.
SSH key sets will have both a public and a private key.

VPN - vireual private network. Allows for connection for a private network
	 over an online connection.
